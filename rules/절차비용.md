# 절차·비용 기준 — 소가·인지액·송달료·관할 산출용

**용도: 작업 보조 도구.** /draft-complaint가 소장 작성 시 인지액·관할 후보를
산출해 **대화창 보고에만** 제시하는 데 쓴다.

운영 규칙:
- 계산 결과는 항상 `[변호사 확인 필요]`를 병기한다. 서면 본문·context JSON에는
  넣지 않는다(제출법원 기재 제외).
- 아래 조문 요지는 전부 MCP 조회로 확인된 것이며, 산출 결과를 대화창에
  보고할 때는 근거 조문에 출처 핀을 병기한다.
- 이 파일의 내용을 서면에 인용하려면 여전히 `리서치/쟁점별_법리.md` 원문 발췌가
  필요하다(인용 단일 진입로 불변).

## 1. 소가 산정

- 재산권 청구: 소로 주장하는 이익 기준(민사소송법 제26조 제1항).
  금전지급청구는 청구금액.
- 병합 청구: 여러 청구의 값을 합산(민사소송법 제27조 제1항). 단, 이자·손해배상·
  위약금·비용이 **부대목적**이면 소가에 불산입(같은 조 제2항 — 원금+지연이자
  청구 시 지연이자는 소가 제외).
- 재산권의 소로서 소가 산출 불능 + 비재산권 목적 소송: **소가 5,000만 원**
  (민사소송등인지규칙 제18조의2 본문). 회사관계 소송 등 같은 조 단서 유형은 1억 원.
- **해고무효확인의 소는 비재산권 소송으로 본다**(민사소송등인지규칙 제15조 제4항)
  → 소가 5,000만 원.
- 1개의 소로 비재산권 소송 + 그 원인 사실에서 발생한 재산권 소송 병합 시:
  액수가 많은 쪽 기준(민사소송등인지법 제2조 제5항. 예: 해고무효확인 + 임금).

## 2. 인지액 (민사소송등인지법 제2조 제1항 — 1심 소장)

| 소가 구간 | 산식 |
|---|---|
| 1천만 원 미만 | 소가 × 50/10,000 |
| 1천만 원 이상 ~ 1억 원 미만 | 소가 × 45/10,000 + 5,000원 |
| 1억 원 이상 ~ 10억 원 미만 | 소가 × 40/10,000 + 55,000원 |
| 10억 원 이상 | 소가 × 35/10,000 + 555,000원 |

- 끝수 처리: 계산액 1천 원 미만이면 1천 원, 1천 원 이상이면 100원 미만 버림
  (같은 조 제2항).
- **전자소송 제출 시 위 금액의 10분의 9**(민사소송등인지법 제16조 제1항).
- 항소장 1.5배, 상고장 2배(같은 법 제3조).
- 검산 예: 소가 5,000만 원 → 50,000,000×0.0045+5,000 = 230,000원
  (전자소송 207,000원). 소가 3억 원 → 300,000,000×0.004+55,000 = 1,255,000원
  (전자소송 1,129,500원).

## 3. 송달료

- 산식 틀: 당사자 수 × 회분 × 1회 송달료 단가.
- 회분·단가는 대법원 예규(송달료규칙의 시행에 따른 업무처리요령) 사항으로
  MCP 조회 범위 밖(`rules/인용규칙.md` §4) → **금액은 항상 `[변호사 확인 필요]`** 처리하고
  산식 틀과 당사자 수만 제시한다. (참고: 민사 제1심 단독·합의사건은 통상
  "당사자수 × 15회분" 관행 — 이 회분 수 자체도 확인 필요로 표기)

## 4. 관할 (1심 제출법원 후보 도출)

후보 도출 순서:
1. **합의관할**(민사소송법 제29조 — 일정한 법률관계, 서면 합의): 계약서에
   관할 합의 조항이 있으면 최우선 검토. 전속적/부가적 여부 해석은
   `[변호사 확인 필요]`.
2. **보통재판적**(제2조 — 피고 주소지): 사람은 주소(제3조), 법인은 주된
   사무소·영업소(제5조 제1항).
3. **특별재판적 — 의무이행지**(제8조, 재산권의 소): 금전채무는 지참채무 원칙상
   채권자(원고) 주소지가 의무이행지가 될 수 있음 — 적용 여부 판단 근거
   (민법상 변제 장소 법리)는 리서치 발췌 후 인용. 그 외 약정 이행지.
4. **관련재판적**(제25조): 병합 청구 중 하나에 관할이 있으면 전부 가능.

산출물: 후보 법원 2~3개 + 각 후보의 근거 조문 핀 + `[변호사 확인 필요 — 관할 선택]`.
구체적 법원 명칭(예: ○○지방법원 ○○지원)의 관할구역 확인은 각급 법원의 설치와
관할구역에 관한 법률 사항 → 미확인 시 "관할구역 확인 필요" 표기.
